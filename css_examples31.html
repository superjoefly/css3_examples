Speed Curves

The following different speed curves are available for transitions:

  • ease: (default) slow start, then fast, then slow end
  • linear: same speed from start to end
  • ease-in: slow start
  • ease-out: slow end
  • ease-in-out: slow start and end
  • cubic-bezier: define your own values
