CSS Examples

This is a collection of CSS Examples that I developed using W3Schools.com. Many of the examples I have here are very similar to the ones on their website. I did this as a learning activity and I'm posting it to exhibit some of the knowledge that I've gained throughout the learning process. Although there are some differences, the markup for these examples is available through W3Schools.

There are three ways to insert a style sheet:

  1. External Style Sheet - uses a separate document with the file name ending in .css; must be linked to the .html document; can be used to style more than one html document.

  2. Internal Style Sheet - defined with the style-element inside the head-section of the HTML document; used to style a single html document.

  3. Inline Styling - uses the style-attribute and applies a unique style to a single element.

The cascading order of the different methods of styling are as follows:

  • Inline style (inside the HTML element)
  • External and internal style sheets (inside the head section)
  • Browser default

An inline style has the highest priority which means that it will override a style defined inside the head tag, or in an external style sheet, or the browser's default value.

Note: if properties for the same selector have been defined in multiple style sheets, the value from the last read (defined) style sheet will be used. If the internal style is defined before the link to the external style sheet, then the exteranl style will be used. If the internal style is defined after the external link, then the internal style will be used.

The individual pages in this project will require a unique styling, therefore, for the purpose of this project I will be mostly using inline and internal css.
