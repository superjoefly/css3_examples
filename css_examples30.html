CSS3 2D Transforms

Translate() Method

The translate() method moves an element from it current position:

This element has been moved (translated) 75px to the right, and 100px down from its original position!










The rotate() method rotates an element clockwise or counter-clockwise:







This element has been rotated 65 degrees clockwise!






The scale() method increases or decreases the size of an element:

This element has had it's size scaled to be two times larger than its original width! Note: the text got wider too!

The skewX() method skews an element along its X axis:

This element has a skewX of 15 degrees!

The skewY() method skews an element along its Y axis:

This element has a skewY of 15 degrees!



The skew() method skews an element along both the x and Y axis.

This element has a negative skew along both the x and y axis!

The matrix() method combines all the 2D Transform methods into one:

This element is transformed using the matrix() method...it uses ALL the 2D Transform methods in one declaration!
