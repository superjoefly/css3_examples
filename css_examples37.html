More flex properties:

  • flex-basis: specifies initial length (along the line) of flex item (example: flex-basis: 200px;)(default value is auto); if a flex basis is not specified, length is according to content.
  • flex-grow: specifies how much a flex item will grow relative to the rest of the flex items (default value is 0).
  • flex-shrink: specifies how much a flex item will shrink relative to the rest of the flex items (default value is 0).

Note: one solution to get equal width flex items is to set flex-grow: 0; and flex-basis: x;

Imprtant: the flex property is shorthand for the flex-grow, flex-shrink and flex-basis properties;

(default value is 0 1 auto;)

Syntax:

flex: flex-grow flex-shrink flex-basis|auto|initial|inherit;
