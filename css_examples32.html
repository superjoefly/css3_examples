Animation Timing Function

The following speed curves are vailable for animations:

  • linear: same speed from start to end
  • ease: (default) slow start, then fast, slow end
  • ease-in: slow start
  • ease-out: slow end
  • ease-in-out: slow start and slow end
  • step-start: steps(1, start)
  • step-end: steps(1, end)
  • steps(int,start|end): a stepping function with two parameters
  • cubic-bezier(n,n,n,n): define your own values (0-1)
  • initial: sets to default value
  • inherit: inherits from parent

Animation Shorthand Property

The animation shorthand property defines multiple animation properties in on declaration:


	animation: 
	name 
	duration 
	timing-function
	delay 
	iteration-count 
	direction
	fill-mode 
	play-state;
