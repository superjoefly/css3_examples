CSS3 Media Queries

Change background-color and text-color:

The following example uses a media query to change the background and text color of the viewport at 800px:

Resize the browser window to see the effect:


	@media all and (min-width: 800px) {
		body {background-color: brown;
		color: beige;
		}
	}

Float menu bar to the left:

On a smaller screen, the menu items will be on top of the content; on a larger screen, they will be to the left of the content.

Resize the browser window to see the effect:


	@media all and (min-width: 600px) {
		  #leftsidebar {
			  width: 200px;
			  float: left;
			  }
		  #main {
			  margin-left: 220px;
			  }
		  }
